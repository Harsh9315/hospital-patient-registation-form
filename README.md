# 🏥 Hospital Patient Registration System

A lightweight and responsive **Web Application** to manage hospital patient admissions. Built as a simple 2-file project using **Node.js, Express.js**, and **HTML5/CSS3** (embedded styling).

---

## ✨ Features
* 📝 **Patient Registration Form:** Input fields for Patient Name, Date of Admission, and Illness Name.
* 📂 **File-Based Database:** Saves records permanently inside a local `student_registry.txt` file using Node's native `fs` module.
* 📋 **Dynamic List View:** Formats and displays the registered patients instantly on a clean, readable layout.
* 🎨 **Embedded CSS UI:** Modern design with smooth inputs, soft shadows, and a professional health-portal color scheme.

---

## 🛠️ Tech Stack Used
* **Backend:** Node.js, Express.js framework
* **Frontend:** HTML5 with embedded CSS3 styling
* **Storage:** Native File System (`fs` module)

---
